# Análisis de Incidente de Seguridad: Exfiltración de Datos en E-commerce

##  Descripción del Proyecto
Este repositorio contiene la documentación detallada y el análisis de un incidente de seguridad real (simulado para fines educativos) ocurrido en una empresa minorista. El objetivo es demostrar habilidades en la fase de **Actividad Post-Incidente** según el marco de trabajo de NIST, incluyendo la identificación de la causa raíz, la cronología del evento y la propuesta de remediación.

##  Habilidades Demostradas
* **Análisis de Logs:** Identificación de patrones anómalos en servidores web.
* **Respuesta a Incidentes:** Comprensión del ciclo de vida de un ataque y gestión de crisis.
* **Documentación Técnica:** Redacción de informes ejecutivos y técnicos para diferentes niveles de la organización.
* **Ciberseguridad Web:** Análisis de vulnerabilidades como la navegación forzada (*Forced Browsing*).

##  Estructura del Repositorio
* `INCIDENT-REPORT.md`: Informe final detallado del incidente.
